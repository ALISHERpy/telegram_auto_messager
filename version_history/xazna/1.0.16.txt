Version: 1.0.16
Release Date: Jul 4, 2024
Release Description:
- Foydalanish qulayligini yaxshilash uchun to'liq qayta ishlab chiqilgan foydalanuvchi interfeysi.
- Shaxsiylashtirilgan tajriba uchun yangi fon rasmi dizaynlari qo'shildi.
- Optimallashtirilgan ishlash va barqarorlik yaxshilandi.
- OFFLINE rejimida xizmatlardan foydalanish imkoniyati optimallashtirildi (Internetga ulanmasdan xizmatlarni boshqarish *930# USSD buyrug'i orqali amalga oshiriladi).
- Depozit hisobvarag'idan summani qisman yechib olish imkoniyati qo'shildi.
-Ilova ichida avtomashinaga ishonchnoma berish imkoniyati qo'shildi.
- Полностью переработан пользовательский интерфейс для улучшения удобства использования.
- Добавлены новые обои оформления для более персонализированного опыта.
- Ускорена производительность и повышена стабильность работы приложения.
- Оптимизирована возможность использование сервисов в OFFLINE режиме (Управление сервисами без доступа в интернет доступно посредством USSD команды *930#).
- Добавлена возможность частичного снятия суммы с депозитного счета
-Добавлена возможность оформления автомобильной доверенности внутри приложения
- Completely redesigned user interface to improve usability.
- Added new wallpaper designs for a more personalized experience.
- Improved performance and increased stability of the application.
- The ability to use services in OFFLINE mode has been optimized (Management of services without Internet access is available via the USSD command *930#).
- Added the ability to partially withdraw the amount from the deposit account
-Added the ability to issue Letter of authority to drive a car within the application
